# 📊 Attendance Calculator

An easy-to-use web-based Attendance Calculator that assists students in calculating how many more classes they need to attend to reach the minimum required attendance percentage.

---

## 🚀 Features

* 📱 Mobile, tablet, and desktop compatible (fully responsive)
* 🧮 Automatically calculates required classes
* ⚡ Simple and fast UI
* 🎯 Easy for students to use

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📌 Working

1. Total number of classes
2. NA classes (Not Applicable)
3. Classes attended
4. Required attendance percentage
5. Calculate

👉 And, you'll know how many more classes you should attend.

1. Clone/download the repository
